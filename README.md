# Quant_Lab — Python 量化交易互動教學平台

![Quant_Lab Banner](https://img.shields.io/badge/Status-Completed-success?style=for-the-badge&logo=react) ![Pyodide](https://img.shields.io/badge/Powered%20by-Pyodide-blue?style=for-the-badge&logo=python) ![PWA](https://img.shields.io/badge/PWA-Ready-f59e0b?style=for-the-badge&logo=pwa)

**Quant_Lab** 是一個基於 **React**、**TypeScript**、**Vite** 與 **Pyodide** 開發的「全前端」Python 量化交易互動教學平台。

你可以直接在瀏覽器中學習理論、編寫、調整參數，並即時執行 Python 量化策略代碼，體驗專業的回測視覺化輸出。完全**無需安裝任何伺服器**或本地環境！

---

## ✨ 平台亮點與技術棧

- **0 伺服器依賴**: 透過 WebAssembly (Pyodide) 直接在前端執行 Python 邏輯、NumPy 運算與完整的事件驅動回測引擎。
- **現代化原子設計 UI**: 
  - 支援一鍵切換**亮色 (Light) / 暗色 (Dark) 高對比主題**。
  - 專業的**終端機 (Terminal)** 模擬輸出與 **CodeMirror** 程式碼編輯器。
  - 將 Python 參數動態綁定至網頁滑桿，所見即所得！
- **豐富的視覺化圖表**: 使用 Chart.js 完美渲染 K 線軌道、均線交叉、乖離率以及資金權益曲線 (Equity Curve)。
- **PWA 支援**: 可作為獨立應用安裝至桌面 (Windows/Mac) 或手機，並支援離線快取，隨時隨地學習量化。

## 📚 漸進式量化學習路徑 (100% 完整適配)

平台精心設計了循序漸進的七大模組，總計 20 個經典必學案例。帶你從最基礎的均線指標，走向進階動態突破與資金管理。強烈建議**跟隨這個路徑**依序學習，效果最佳：

### 🔰 模組一：量化入門 (奠定基礎)
量化交易的出發點。我們從最廣為人知的指標開始，建立對「回測機制」與「交易訊號」的基本認知。
- **[1.1] 雙均線策略**: 認識量化交易。學習短均線與長均線的金叉與死叉買賣訊號，以及基礎的回測引擎觀念。

### 📈 模組二：趨勢跟蹤 (過濾與自適應)
學會均線後，你會發現它在橫盤震盪市中容易被雙巴虧損。這一個模組教你如何利用更高級的數學與指標來「認清趨勢」並「過濾雜訊」。
- **[2.1] 經典 MACD 策略**: 利用平滑異同移動平均線的零軸穿越來捕捉市場趨勢。
- **[2.2] ADX 輔助 MACD**: 導入趨勢強度指標 (ADX)，過濾無方向的震盪市假訊號，解決 MACD 只會追漲殺跌的弱點。
- **[2.3] 自適應均線 (AMA)**: 均線太遲鈍怎麼辦？根據市場當下波動率，自動動態調整平滑係數的均線系統。
- **[2.4] 阿隆指標 (AROON)**: 不看均線，改從「距離上一次創新高的時間」來捕捉趨勢發韌初期的領先指標。
- **[2.5] 簡易波動 EMV**: 將成交量 (Volume) 與價格波幅結合，找出阻力最小的動能趨勢。

### ⚡ 模組三：突破策略 (尋找壓力與支撐)
當大趨勢發動時，往往伴隨著關鍵價位的突破。本模組會帶你尋找各種被歷史驗證過有效的動態 / 靜態邊界。
- **[3.1] Dual Thrust 區間突破**: 華爾街名噪一時的經典日內突破策略，根據前 N 日震幅動態推算今日的開盤上下軌操作區間。
- **[3.2] 日內高低點突破**: 捕捉開盤初期的劇烈市場情緒方向，突破今日最高點即追漲，跌破最低即殺跌。
- **[3.3] 增強版唐奇安通道**: 傳奇「海龜交易法則」的核心。利用突破 N 日極值來建立長線部位，並實作動態的停利止損線。
- **[3.4] Hans123 日內突破**: 外匯市場常用的開盤時段標竿突破策略。
- **[3.5] 菲阿里四價**: 日本傳奇操盤手的經典防線策略，運用前一日的 OHLC (開高低收) 建立今日的不可越雷池。
- **[3.6] 動態波幅突破**: 利用時間序列的波動率 (ATR) 自動伸縮突破通道，防範震盪期的假突破。
- **[3.7] R-breaker 交易策略**: 高手境界。結合了「趨勢突破進場」與「逆勢均值回歸出場」的複雜網格型態高階交易策略。

### 🔄 模組四：切換策略 (應對不同市況)
既然我們學了趨勢策略（第二章）與震盪策略，那如果市場環境突然改變了怎麼辦？
- **[4.1] 經典恆溫器策略**: 使用 CMI (Choppiness Market Index) 來區分這陣子是牛市還是猴市，從而自動切換「順向突破追擊」與「逆勢低買高賣」的雙核心邏輯大腦。

### ⚖️ 模組五：高級交易與套利 (擺脫大盤單邊風險)
跳脫單純的預測漲跌，尋找相關市場中的相對不平衡點。
- **[5.1] 期貨跨期套利實戰**: 從單邊投機走向穩健對沖。尋找高度相關資產的「價差擴張與收斂」，無論大盤暴漲暴跌都能穩定抽佣。
- **[5.2] 乖離率 BIAS 策略**: 尋找物極必反的均值回歸點，在市場極度恐慌超賣或貪婪超買時反向操作。

### 🛡️ 模組六：風險與資金管理 (生存的關鍵)
尋找買賣點只是交易的一小部分，部位管理 (Position Sizing) 才是決定長期是獲利暴雪還是走向破產的分水嶺。
- **[6.1] 馬丁格爾策略與倍增法**: 探討賭徒謬誤。寫程式親眼見證深不見底的「越跌越買、輸了攤平加倍」逆勢凹單破產軌跡。
- **[6.2] 凱利公式部位管理**: 資金管理決定生存機率。透過蒙地卡羅並發模擬，測試不同下注比例（原版滿倉凱利、半凱利保守型）對長期資產增長的劇烈影響。
- **[6.3] 反向馬丁格爾 (逆勢倍增法)**: 投資大師推崇的「截斷虧損，讓獲利奔跑」，只在有獲利底氣保護時才順勢加碼的聖杯操作。

### 📊 模組七：終極實戰
- **[7.1] 基本面量化實踐**: 擺脫單純的 K 線圖與技術指標，結合 EPS、PE、ROE 等財報表現進行多因子共振的股票選擇與換股模型。

## 🕹️ 平台使用方式

1. 從左側的導覽列由上往下選擇**教學單元**。
2. 閱讀中央面板的**內容說明與核心理論**，了解策略背後的數學邏輯與優缺點。
3. 調整右上方的**參數拉桿 (Parameters)**，你會發現下方的 Python 代碼也會同步自動動態更新變數！
4. 點擊右上角的 **「▶ Run」**，等待 WebAssembly 在你的瀏覽器內瞬間算完回測。
5. 切換至 **Terminal Result 面板** 查看詳細的逐筆交易紀錄。
6. 在中央的 **執行結果 面板** 查看包含獲利率、勝率、夏普比率 (Sharpe Ratio) 與精美的 Chart.js 動態分析圖表。

## 🛠️ 本地開發與部署

如果你想在本地端執行開發伺服器，或自行修改增建更多策略：

```bash
# 1. 安裝依賴庫
npm install

# 2. 啟動 Vite 本地開發伺服器
npm run dev

# 3. 建置生產環境版本 (全靜態網頁，可直接部署於 GitHub Pages / Vercel / Netlify)
npm run build
```

*(註：本互動教學平台僅供學習與交易邏輯研究使用，模擬引擎與真實市場存在滑點、流動性與手續費差異，請勿直接將代碼用於實盤自動交易。)*
