# Python 期貨量化交易實戰 - 策略代碼清單

本倉庫包含了多個基於 **FMZ (BotVS)** 量化交易平台的 Python 範例策略。涵蓋了從基礎指標應用到複雜的日內交易、套利策略以及資金管理模型。

## 🚀 快速開始

這些策略代碼專為 **FMZ 量化平台** 設計，可以直接在 FMZ 平台上點擊「添加策略」並貼上代碼運行。

- **平台網址**: [FMZ.COM](https://www.fmz.com)
- **程式語言**: Python 3
- **依賴庫**: `fmz` (內建), `numpy`, `talib` (部分策略需要)
