# 量化交易互動教學平台 (pwa_quant_python)

本倉庫包含了一個基於 **Pyodide** 與 **PWA (漸進式網頁應用)** 技術開發的「全前端」Python 量化交易互動教學平台。你可以直接在瀏覽器中編寫、修改並執行 Python 量化策略代碼，無需安裝任何伺服器或本地環境。

除了最新的 PWA 互動平台代碼（位於 `pwa_quant_python` 目錄），本倉庫也妥善保存了之前基於 **FMZ (BotVS)** 平台的原始 Python 範例碼（位於 `Python_code` 目錄）。

## 🚀 平台特色與技術棧

- **0 伺服器依賴**: 透過 WebAssembly (Pyodide) 直接在前端執行 NumPy 與 Python 回測邏輯。
- **即學即練**: 內建 CodeMirror 程式碼編輯器與 Chart.js 圖表庫，視覺化呈現資金曲線與副圖指標。
- **PWA 支援**: 可作為獨立應用安裝至桌面或手機，並支援離線快取。
- **靜態部署**: 可完美託管於 GitHub Pages。

## 📚 課程模組總覽

平台將量化交易策略分為多個從基礎到進階的漸進式模組，目前所有的策略代碼皆已納入前端網頁模組中：

### 模組一：量化入門
- **[雙均線策略]**: (💯 已完整適配) 從最經典的量化交易入門策略開始。學習短均線與長均線的金叉與死叉如何產生買賣訊號，以及如何使用 Python 來建立基礎的回測引擎觀念。

### 模組二：趨勢跟蹤
- **[MACD 策略]**: (💯 已完整適配) 利用 MACD (平滑異同移動平均線) 的快慢線交叉與零軸穿越來捕捉市場趨勢。學習如何繪製客製化的柱狀圖(Histogram)並利用動能反轉進行交易。
- ⏳ *等待重構中的原始碼*: ADX輔助MACD、自適應均線(AMA)、阿隆指標(AROON)、簡易波動EMV。

### 模組三：突破策略
- **[Dual Thrust 日內突破]**: (💯 已完整適配) 過去在華爾街名噪一時的經典區間突破策略。學習計算前 `N` 日的高低點範圍，動態推算今日的開盤上下軌操作區間。
- **[增強版唐奇安通道]**: (💯 已完整適配) 經典且聲名大噪的「海龜交易法則」核心。根據一段時間內的高低點建立通道，學習利用突破極值來追蹤趨勢並實作停利止損線。
- ⏳ *等待重構中的原始碼*: 日內高低點突破、Hans123日內突破、菲阿里四價、動態波幅突破、R-breaker交易策略。

### 模組四：切換策略
- ⏳ *等待重構中的原始碼*: 經典恆溫器策略 (使用 CMI 指標區分趨勢與震盪市場自動切換邏輯)。

### 模組五：高級交易與套利
- ⏳ *等待重構中的原始碼*: 期貨跨期套利策略、乖離率BIAS策略。

### 模組六：資金管理
- **[凱利公式驗證]**: (💯 已完整適配) 資金管理決定了在市場上的生存能力。透過蒙地卡羅多線程模擬，測試並視覺化不同的下注比例（原版凱利、半凱利、10%、50%）對長期資產成長的影響。
- ⏳ *等待重構中的原始碼*: 正向馬丁格爾、反向馬丁格爾策略。

### 模組七：數據獲取
- ⏳ *等待重構中的原始碼*: 獲取基本面數據演示。

## 🕹️ 平台使用方式

1. 點擊進入任一**課程模組單元**。
2. 閱讀左側的**原理解說**，了解策略背後的數學邏輯與優缺點。
3. 調整右側的**策略參數拉桿**（這會自動同步更新到你的 Python 代碼中）。
4. 檢視或自由修改下方的 **Python 策略代碼** (`strategy.py`)。
5. 點擊 **「▶ 執行」**，等待 WebAssembly 計算出回測結果。
6. 互動式檢視生成的**詳細績效數據**與 Chart.js **視覺化 K 線圖/權益曲線**。
